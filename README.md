# bot-airdrop

### Install dependencies
```
yarn
// or
npm i
```

### Generate seed phrase
```
yarn generate:mnemonic
// or
npm run generate:mnemonic
```

### Generate 100 private keys
```
yarn generate:keys
// or
npm run generate:keys
```

### Send native token to 100 addresses
```
yarn send:native
// or
npm run send:native
```

### Send USDT through 100 addresses
```
yarn send:loop
// or
npm run send:loop
```
